Important Notes:

To view the included components, see the following section.

Both the Visual C++ 2008 SP1 Redistributable and the DirectX End-User Runtimes must be installed prior to running setup. Most systems should already have these installed, but in the event that they are not, they are available through the links below.

-Visual C++ 2008 SP1 Redistributable
https://www.microsoft.com/en-us/download/details.aspx?id=26368
-DirectX End-User Runtimes (June 2010)
https://www.microsoft.com/en-us/download/details.aspx?id=8109

A fully Direct3D 9-compatible system is required for the essential components to properly function.

Windows 2000 or above is required.

The supported game localizations are English, German, and French. Setup should automatically detect the appropriate language.

To install mods, place the files wherever you wish and add either the full or relative paths to their locations in cam_mod.ini, which is located in your install directory.

To uninstall or install additional components, simply run the patcher again and select the desired combination.

Do not set any compatibility modes on the game executable or install the game to a protected directory such as Program Files.

To install the game from the original CDs, check the box on the directory selection page and follow any additional instructions.
